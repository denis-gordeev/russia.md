# Document-Signature Integration Notes

## Focus Areas

- Signing is not the same thing as document generation.
- Approval order, signer identity, and archive evidence must be explicit.

## Practical Guidance

- Keep draft preparation reversible until the final review checkpoint.
- Record which documents need certificate or signer validation before the signing ceremony.
- Preserve signed artifacts and their metadata together.
- Surface where a user still needs to leave the agent flow and complete the signature in an approved interface.

## Agent Boundaries

- Agents can prepare signature packets, check prerequisites, and describe review order.
- Agents should not claim that a document is legally complete until the signing system confirms it.
